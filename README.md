# DigitRecognition  
手書き文字認識アプリ 

Androidアプリとして、kotlinで制作。  
認識機能は、TensorFlowLiteを利用。  
学習データはMnist-Digitからです。  

構成は
-メインアクティビティクラス  
-お絵描き用のカスタムビュークラス  
-識別用のクラス  
から成ります。
